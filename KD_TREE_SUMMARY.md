## 🚀 快速開始

### 基本使用

```python
from src.utils.kdtree import KDTree, Point

# 1. 建立點
points = [
    Point([2, 3], "A"),
    Point([5, 4], "B"),
    Point([9, 6], "C"),
]

# 2. 構建樹
tree = KDTree(points)

# 3. 搜索
target = Point([6, 3])
nearest, distance = tree.find_nearest(target)
```

## 📚 應用場景

### 推薦場景
✅ **適合使用 KD-Tree：**
- 2-5 維空間的最近鄰搜索
- 動態點較少的應用
- 需要快速查詢的系統
- 實時推薦系統

❌ **不適合使用 KD-Tree：**
- 10+ 維高維數據
- 頻繁的大量插入/刪除
- 需要精確距離度量的地理應用（使用球面距離）

## 📞 常見問題

**Q：為什麼樹總是平衡的？**
A：因為我們選擇中位數作為分割點，確保左右子樹大小平衡。

**Q：如何處理重複點？**
A：在距離相等時，會返回先找到的點。可修改比較邏輯來自定義行為。

**Q：高維數據為什麼性能下降？**
A：這是「維度詛咒」現象。維度增加時，點在空間中分佈稀疏，性能接近暴力搜索。
